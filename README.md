## 🏗 Development Guide

### 1. clone the repository

```sh
git clone https://github.com/rahulrajdahal/snake-game-c-sharp.git
```

### 2. Navigate to the directory

#### npm

```sh
cd snake-game-c-sharp
```

### 3. Build the game

```sh
dotnet build
```

### 4. Run the game

```sh
dotnet run
```
